# PVProject

Go to https://git-scm.com/downloads
Download and install Git for your operating system (Windows, macOS, or Linux).
During installation, leave most options as default.

Verify installation: git --version

Open your VS Code terminal (Ctrl + ~) and run:
git config --global user.name "Your Name"
git config --global user.email "your_email@example.com"
git config --list

To clone: git clone https://github.com/Tanima-Sharma/PVProject.git

Common Commands, you can use:
git status
git pull
git add .
git commit -m "Your Comment"
git push
